SyncDrive

SyncDrive est une application de cloud personnel propulsé par Electron. Elle permet de stocker et de partager des fichiers via Google Drive de manière accessible depuis n'importe où. Le but est de pouvoir récupérer son travail ou une sauvegarde de jeu facilement entre plusieurs ordinateurs et ce même d'OS différents.

L'Histoire du Projet

SyncDrive marque le début de mon aventure dans le développement assisté par l'IA. C'est mon tout premier projet réalisé à l'aide de l'IA (Bolt.new en l'occurence).

Qu'est-ce que Bolt ?
Bolt est un agent de développement web IA avancé qui permet de créer, éditer et déployer des applications full-stack directement dans le navigateur. Il m'a permis de transformer rapidement mes idées en une application fonctionnelle, en gérant la complexité technique tout en me laissant le contrôle sur la logique métier et le design.

Technologies Utilisées

  • React & Vite pour le frontend
  • Node.js pour le backend
  • Base de données SQL
  • Design responsive moderne

Fonctionnalités Clés

📂 Gestion de Fichiers

Upload, organisation et gestion intuitive de vos documents, images et médias dans le cloud.

🔒 Sécurité Avancée

Système d'authentification robuste et protection des données pour garantir la confidentialité.

🔄 Synchronisation

Accès instantané à vos fichiers depuis n'importe quel appareil, à tout moment.

🎨 Interface Moderne

Une expérience utilisateur fluide et agréable, conçue avec soin pour une prise en main immédiate.
